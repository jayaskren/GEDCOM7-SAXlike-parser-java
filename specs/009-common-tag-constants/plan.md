# Implementation Plan: Common Substructure Tag Constants

**Branch**: `009-common-tag-constants` | **Date**: 2026-03-07 | **Spec**: [spec.md](spec.md)
**Input**: Feature specification from `/specs/009-common-tag-constants/spec.md`

## Summary

Add 16 new nested classes and ~70 new `public static final String` constants to `GedcomTag` for depth-2+ GEDCOM 7 substructure tags that currently lack constants. This covers structures like PLAC (MAP, FORM, LANG), MAP (LATI, LONG), DATE (TIME, PHRASE), ADDR (CITY, STAE, POST, CTRY), NAME (GIVN, SURN, NICK), FILE (FORM, TITL), and others. No existing constants or classes are modified.
